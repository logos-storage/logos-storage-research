# Privacy and Anonymity Research

This directory contains the research outcome relating to privacy and anonymity of logos-storage file-sharing.

Currently it contains the following:

## [Analysis of the file sharing components](./analysis-of-file-sharing-components.md)

The aim of this research is to take a deeoer look into the components of the file sharing client identifying what information are leaked during the file sharing process (both download and publish) which require either changing to protocol or enhancing the protocol with privacy-preserving schemes such as mixnet (libp2p-mix), onion routing, PIR, etc.
